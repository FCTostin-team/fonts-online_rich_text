A specialized web tool for generating "Rich Text" tags used in Factorio. It allows players to easily create formatted text strings with specific colors and in-game fonts (like `default-bold`, `compilatron`, etc.) for use in chat, train station names, map markers, and blueprints.

## Features

* **Visual Font Selector**: Features a dropdown with visual previews of all available Factorio fonts (e.g., `heading-1`, `count-font`, `default-game`) to see exactly how the text style looks.
* **Rich Text Generation**: Automatically formats text into the correct `[font=...][color=...][font]` syntax supported by the game engine.
* **Color Picker**: Built-in HEX color selector to apply custom coloring to the text.
* **Length Validation**: Real-time character count monitoring to ensure the generated string fits within game limits (e.g., for map tags).
* **Multilingual UI**: Supports 12 languages including English, Russian, Ukrainian, German, and Chinese.

## Installation and Usage

1.  Clone the repository or download the source files.
2.  Open `index.html` in any modern web browser.
3.  Type your desired text and select a font style from the visual list.
4.  Pick a color and click **"Copy"** to get the formatted code for Factorio.
